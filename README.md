# Auckland Car Glass

Static website for **aucklandcarglass.co.nz** — plain HTML/CSS, no build step.

- **Hosting:** Netlify, deployed from this repo (publish directory = repo root, see `netlify.toml`).
- **Contact form:** uses Netlify Forms (no third-party service).
- **Redirects:** retired/renamed pages handled in `_redirects`.
- **SEO:** `sitemap.xml` and `robots.txt` are included.

## Updating
Edit the files, commit, and push. Netlify redeploys automatically.
